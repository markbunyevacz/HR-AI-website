|  | II. Community Felállításához és Életben Tartásához Szükséges User Storyk |  |
|  | A közösség (community) célja a certifikáció birtokosainak exkluzív szakmai hálózatának biztosítása, a tudásmegosztás és az önszerveződés támogatása. |  |
|  | Részterület | User Story (Felhasználói Igény) |
|  | Közösségi Tudásmegosztás | Mint certified szakember, szeretnék cikkeket és esettanulmányokat publikálni a fórumon (akár AI támogatásával), hogy megosszam a tapasztalataimat a közösséggel. |
|  |  | Mint tag, szeretnék szavazni a következő hetek vagy hónapok közösségi témáiról (vagy két hetente új témát bedobni), hogy irányítottan gondolkodhassunk és beszélgethessünk a legaktuálisabb AI hatásokról. |

|  |  | Mint tag, szeretnék élő online eseményeken (workshop/networking) részt venni hetente kétszer – egy magyarázó és egy kiscsoportos beszélgetés formájában – hogy a tudásomat elmélyítsem és új kapcsolatokat építsek. |
|  | III. A Certificate Megszerzéséhez Szükséges User Storyk (CAHIS™) |  |
|  | A tanúsítvány megszerzése egy 6-9 hónapos, strukturált programot igényel, amely elméleti tudás, gyakorlati készségfejlesztés és értékelés kombinációja. |  |
|  | Részterület | User Story (Felhasználói Igény) |
|  | Képzési Modulok | Mint jelölt, szeretnék egy modult elvégezni az AI technológiák és a munkaerőre gyakorolt hatásuk megértéséről (pl. a rule-based automation és a machine learning közötti különbségről). |
|  |  | Mint jelölt, szeretnék tanulni az emberi pszichológia és a változással szembeni ellenállás gyökereiről az AI adoption kontextusában, ideértve a Kübler-Ross gyászgörbét. |
|  |  | Mint jelölt, szeretnék elsajátítani egy standardizált Human Impact Assessment módszertant és templétet, amely magában foglalja a stakeholder interjúk és dolgozói survey-k tervezését. |
|  |  | Mint jelölt, szeretnék gyakorlati képzést kapni a C-level kommunikációról és befolyásolásról (pl. hogyan javasoljak alternatív megközelítéseket vagy vétózzak meg projekteket). |
|  |  | Mint jelölt, szeretnék gyakorlati tudást szerezni az AI transzformációs change management executionről (belső marketing kampányok, kommunikációs cascade-ek, training programok tervezése). |
|  |  | Mint jelölt, szeretnék valós élethelyzetekről szóló case study-kat elemezni (Klarna, IBM, Duolingo), hogy megértsem, mi működött és mi nem. |
|  | Értékelés és Vizsgáztatás | Mint jelölt, szeretném bemutatni egy valós vagy realisztikusan szimulált AI projektre készült teljes Human Impact Assessment riportot, amelyet panel előtt védhetek meg. |
|  |  | Mint jelölt, szeretnék részt venni egy írásbeli vizsgán és szimulált case study-n, amely teszteli a teoretikus tudásomat és gyakorlati döntéshozatali képességemet. |
|  |  | Mint jelölt, szeretnék role-play gyakorlatokon részt venni, ahol C-szintű vezetőket kell tanácsolnom nehéz etikai és stratégiai döntési pontokon. |
|  | Elismerés | Mint jelölt, szeretnék fizikai formájú oklevelet és giftet (pl. asztaldísz, papírnehezék) kapni, amelyet irodában kihelyezhetek és posztolhatok a LinkedInre, hogy a HR-közösségben látható legyen az eredményem. |
|  | IV. A Certificate Életben Tartásához Szükséges User Storyk (Megújítás/Folyamatos Fejlődés) |  |
|  | A CAHIS™ tanúsítványt három évenként meg kell újítani, amelyhez folyamatos szakmai fejlődés (Continuing Professional Development – CPD) igazolása szükséges. |  |
|  | Részterület | User Story (Felhasználói Igény) |
|  | Certifikáció Megújítás (Renewal) | Mint certified professional, szeretném igazolni a Continuing Professional Development (CPD) pontokat/egységeket (PDUs), hogy három évenként megújíthassam a tanúsítványomat. |
|  | Pontszerzés Publikációval | Mint certified professional, szeretnék pontokat kapni (pl. 10-15 pontot) a közösségi platformon publikált cikkekért/esettanulmányokért, amelyek hozzájárulnak a certifikációm éves szintű életben tartásához. |
|  | Nyilvános Verifikáció | Mint a tanúsítvány birtokosa, szeretnék szerepelni egy nyilvános, ellenőrizhető regiszterben (registry), hogy potenciális munkáltatók vagy kliensek igazolni tudják az érvényes CAHIS™ minősítésemet, ami hitelességet ad a piacon. |
|  | Közösségi Tudásmegosztás és Networking | Mint certified professional, szeretnék élő online eseményeken (networking, kiscsoportos beszélgetések) részt venni hetente kétszer, hogy a tudásomat elmélyítsem, és kapcsolatokat építsek más, hasonlóan képzett szakemberekkel. |
|  | Témavezérelt Párbeszéd | Mint közösségi tag, szeretném, ha a közösség moderátorai kéthetente új témát dobnának fel, amely irányítja a gondolkodást az aktuális AI hatásokról, de lehetőségem van szavazni, vagy felülírni a topikot az újdonságok (pl. új GPT-k) miatt. |
|  | Önszerveződés (Board) | Mint senior tag, szeretnék részt venni a közösség önszerveződésében (mint egy board / gittegylet), hogy az adminisztráció és a szervezet működtetése a tagok által történjen. |
